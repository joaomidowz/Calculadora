
# Calculadora em JS

Uma calculadora feita em uma aula da OneBitCode com o intuito de aprendizado sobre JS e estilização com css.


## Funcionalidades

- Operações básicas
- Resultado instatâneo
- Inerativo com o click
- Temas dark e light
- Funcionalidade de Clear e dar Copy & Past
